Данный проект представляет собой систему автоматического распознавания автомобильных номеров (ALPR) для 8 стран СНГ: Россия (RU), Казахстан (KZ), Беларусь (BY), Армения (AM), Азербайджан (AJ), Таджикистан (TJ), Узбекистан (UZ), Кыргызстан (KG).

Система состоит из двух этапов:

Детекция номера — YOLO11n находит номерной знак на изображении автомобиля.

Классификация страны — MobileNetV2 определяет страну-владельца по кропу номера.

  Возможности
Детекция номерных знаков на изображениях с помощью YOLO

Классификация страны-владельца номера (8 классов)

Извлечение кропов номеров из изображений

Аугментация данных для улучшения качества обучения

Визуализация результатов с отображением рамок и подписей
